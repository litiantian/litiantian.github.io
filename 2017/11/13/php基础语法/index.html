php基础语法

php中文名为“超文本预处理器”，是一门后台语言。由于他简单，容易入门，它是在web开发中使用最广泛的一种编程语言。
下面是他的基础语法，自学php的小白可以参考参考；

####1.变量定义使用$符号,php三种语法格式：
<?php 代码块 ?>
<? ?> php短标签，但是这个不提倡使用，因为它取决于服务器配置
<% %> sap,更加不提倡
php每句话的末尾必须有;分号
echo: 输出变量到浏览器
print_r: 输出数组

####2.传递给后台php的参数使用？连接
?后面的参数得是get类型

####6. 函数

  • 语法（和js相似）
    注意：形参可以设置默认值,具有默认值的排在无默认值的右边

    function myfunc($param2,$param = “ltt”){

    echo $param." hello world";

    }
    myfunc();

  • 函数的闭包
    js函数有闭包,php中函数也有闭包使用 use(变量)来保存函数内部的局部变量

    function getFunc(){

    $array = array();
for($i = 0 ; $i < 10 ; $i++){
    $array[$i] = function()use($i){
        return 10 + $i;
    };    
};
return $array;

    }

  • 变量的作用域
    函数体内部定义的变量只在函数体内有效，在函数体内部可以使用global关键字将变量定义为全局变量。
    下面的代码不加global $a 时，执行后打印$a还是为10，而加了之后打印$a为7

    $a = 10;
    function addFunc($a){

    global $a;
echo $a = $a - 3;

    }
    addFunc(10);
    echo $a;

  • 引用传值
    在形参前面添加&符号，传的是变量本身，而不是它的值
    下面代码打印的$b的值为9

    function myFunc_2(&$a){

    $a--;
echo $a;

    }
    $b = 10;
    myFunc_2($b);
    echo $b;

  • 内置函数

    enpty()判定一个变量是不是存在,如果变量不存在返回true
    第一种情况变量没有定义，返回的true;
    第二种：$luxp = “”; 即使定义了，但是值为空值也返回true;
    第三种：$lup = 0,”0”,null; 返回true;
    总结：empty()只有变量再定以后，并且赋值，且值不为空，0，”0”，null，返回false

isset()检测变量是否已设置并且非 NULL
__总结:当变量为null时，返回值为false，其余返回true;
